# Agent Guide

This is a simple frontend CRUD application built with vanilla HTML/CSS/JavaScript.

## Project Structure
- `index.html` — Main page
- `css/` — Stylesheets
- `js/` — JavaScript modules
- `docs/` — Project documentation

## Important Rules
- Use vanilla HTML/CSS/JavaScript only. No frameworks (React, Vue, etc.).
- Use ES6 module syntax (`import/export`).
- All data is stored in localStorage (no backend).
- Follow the coding style in `docs/coding-style.md`.
- Follow the component patterns in `docs/components.md`.
