# Components

## Item List
- Render items as `<ul>` with `<li>` elements
- Each item shows: title, description, created date
- Action buttons: Edit, Delete

## Form
- Single form for both Create and Edit
- Fields: title (required), description (optional)
- Submit button text changes based on mode: "Add" vs "Update"
- Cancel button to reset form

## Notification
- Show success/error messages as a toast at the top
- Auto-dismiss after 3 seconds
